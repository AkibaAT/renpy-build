Ren'Py Nightly Builds

The documentation for the latest nightly can be found here.

These are nightly builds of Ren'Py, primarily created to let you help with testing and development of Ren'Py. APIs introduced in nightly builds may not be supported in a final release of Ren'Py.

If, while testing these builds, you find a bug in Ren'Py, please let us know by filing a bug at https://github.com/renpy/renpy/issues.

These builds are made on a best-effort basis, on computers that can be used for other projects. A day's build may be missing if the master git branch can't be built, or if the computers that build it are not available. Please email pytom@bishoujo.us if a build you're expecting is missing - I'll be able to give you a better status update.
